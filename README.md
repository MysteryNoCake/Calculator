# Calculator
# Расчет ФОТ 2026

Калькулятор расчета фонда оплаты труда.

## 🚀 Установка и запуск

1. Скачайте файл `#####.exe`
2. Запустите файл двойным кликом
3. Загрузите Excel файл с данными
4. Нажмите "Рассчитать"

## 📝 Использование

- **Загрузить файл** — загрузка Excel с данными сотрудников
- **Добавить изменение** — редактирование параметров сотрудника
- **Рассчитать** — расчет ФОТ
- **Сохранить** — экспорт результатов в Excel

## 🔧 Создание ярлыка

1. Кликните правой кнопкой на `#####.exe`
2. Выберите "Создать ярлык"
3. Готово! Иконка автоматически применится

## 📋 Требования

- Windows 7 и выше
- Не требует установки Python
